## рҹҡҖ Getting Started

### Run locally (requires Node.js)
```bash
git clone https://github.com/YOUR_USERNAME/promises-worship.git
cd promises-worship
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000)

### Deploy
Push to GitHub вҶ’ auto-deploys on Vercel.
